# 台股即時交易系統 V 1.0

一個使用 PySide6 開發的台灣股票交易模擬應用程式，整合台灣證券交易所（TWSE）的即時股價 API。

## 功能特色

- 📈 **即時股價顯示** - 整合台灣證券交易所 API，顯示真實的股票價格和市場數據
- 🔍 **股票查詢** - 支援輸入股票代碼查詢台股資訊
- 💹 **模擬交易** - 買進/賣出功能，包含手續費和證交稅計算
- 📊 **持倉管理** - 顯示當前持倉明細和市值
- 📝 **交易記錄** - 完整的交易歷史記錄
- 💰 **帳戶資訊** - 即時顯示現金、總資產等資訊
- 🎨 **現代化UI** - 類似「投資先生」的專業交易介面設計，採用模組化架構

## 專案結構

專案採用模組化設計，每個檔案不超過 300 行，易於維護：

```
PySide6/
├── main.py              # 主程式入口（236行）
├── stock_api.py         # 股票 API 模組（145行）
├── account.py           # 帳戶系統模組（94行）
├── ui_components.py     # UI 組件模組（299行）
├── styles.py            # 樣式定義模組（173行）
└── requirements.txt     # 依賴套件清單
```

### 檔案說明

- **main.py** - 主程式，負責整合所有模組並創建應用程式視窗
- **stock_api.py** - 處理台灣證券交易所 API 的呼叫和資料解析
- **account.py** - 帳戶系統，處理交易邏輯、持倉管理和交易記錄
- **ui_components.py** - 可重用的 UI 組件（搜尋、價格顯示、交易、表格等）
- **styles.py** - 統一管理所有 UI 樣式和主題

## 使用說明

### 查詢股票
- 在「股票查詢」區域輸入股票代碼（例如：2330 代表台積電）
- 點擊「查詢」按鈕或按 Enter 鍵
- 系統會自動顯示該股票的即時行情資訊

### 買進股票
1. 在「買進」區域設定要購買的股數
2. 點擊「買進」按鈕
3. 系統會自動計算手續費（0.1425%）並扣除現金

### 賣出股票
1. 在「賣出」區域設定要賣出的股數
2. 點擊「賣出」按鈕
3. 系統會自動計算手續費（0.1425%）和證交稅（0.3%）並增加現金

### 查看資訊
- **帳戶資訊**：顯示當前現金和總資產
- **持倉明細**：顯示所有持有的股票、股數、現價和市值
- **交易記錄**：顯示所有交易歷史，包括時間、類型、價格、數量等

## API 資料來源

本應用程式使用以下 API：

1. **台灣證券交易所（TWSE）API**
   - 個股日成交資訊：`https://www.twse.com.tw/exchangeReport/STOCK_DAY`
   - 個股即時資訊：`https://mis.twse.com.tw/stock/api/getStockInfo.jsp`

## 模組化設計優勢

- ✅ **易於維護** - 每個檔案不超過 300 行，邏輯清晰
- ✅ **可重用性** - UI 組件可在其他專案中重複使用
- ✅ **易於測試** - 各模組功能獨立，方便單元測試
- ✅ **易於擴展** - 新功能可作為新模組添加，不影響現有程式碼

## 注意事項

- ⚠️ 本應用程式為**模擬交易系統**，所有交易均為虛擬模擬，不會產生實際的金錢交易
- 📡 API 資料更新頻率為每 5 秒一次
- 💡 交易手續費和證交稅按照台灣股市實際費率計算
- 🌐 需要網路連線才能獲取即時股價資料
