# 投資先生 - 台股即時交易系統

一個使用 PySide6 開發的台灣股票交易模擬應用程式，整合台灣證券交易所（TWSE）的即時股價 API。

## 功能特色

- 📈 **即時股價顯示** - 整合台灣證券交易所 API，顯示真實的股票價格和市場數據
- 🔍 **股票查詢** - 支援輸入股票代碼查詢台股資訊
- 💹 **模擬交易** - 買進/賣出功能，包含手續費和證交稅計算
- 📊 **持倉管理** - 顯示當前持倉明細和市值
- 📝 **交易記錄** - 完整的交易歷史記錄
- 💰 **帳戶資訊** - 即時顯示現金、總資產等資訊
- 🎨 **現代化UI** - 類似「投資先生」的專業交易介面設計

## 系統需求

- Python 3.8 或以上版本
- Windows / macOS / Linux

## 安裝步驟

1. **安裝依賴套件**
   ```bash
   pip install -r requirements.txt
   ```

2. **執行應用程式**
   ```bash
   python main.py
   ```

## 使用說明

### 查詢股票
- 在「股票查詢」區域輸入股票代碼（例如：2330 代表台積電）
- 點擊「查詢」按鈕或按 Enter 鍵
- 系統會自動顯示該股票的即時行情資訊

### 買進股票
1. 在「買進」區域設定要購買的股數
2. 點擊「買進」按鈕
3. 系統會自動計算手續費（0.1425%）並扣除現金

### 賣出股票
1. 在「賣出」區域設定要賣出的股數
2. 點擊「賣出」按鈕
3. 系統會自動計算手續費（0.1425%）和證交稅（0.3%）並增加現金

### 查看資訊
- **帳戶資訊**：顯示當前現金和總資產
- **持倉明細**：顯示所有持有的股票、股數、現價和市值
- **交易記錄**：顯示所有交易歷史，包括時間、類型、價格、數量等

## API 資料來源

本應用程式使用以下 API：

1. **台灣證券交易所（TWSE）API**
   - 個股日成交資訊：`https://www.twse.com.tw/exchangeReport/STOCK_DAY`
   - 個股即時資訊：`https://mis.twse.com.tw/stock/api/getStockInfo.jsp`
   - 所有股票資訊：`https://www.twse.com.tw/exchangeReport/STOCK_DAY_ALL`

## 注意事項

- ⚠️ 本應用程式為**模擬交易系統**，所有交易均為虛擬模擬，不會產生實際的金錢交易
- 📡 API 資料更新頻率為每 5 秒一次
- 💡 交易手續費和證交稅按照台灣股市實際費率計算
- 🌐 需要網路連線才能獲取即時股價資料
